# OMR Answer Sheet

브라우저에서 바로 사용할 수 있는 **OMR(객관식 답안지) 웹앱**입니다.  
시험 시간 측정, 답안 마킹, 정답 입력/채점, 오답·미답 확인, 문항별 소요 시간 분석까지 한 화면에서 할 수 있습니다.

***

## 어떻게 생긴 서비스인가요?

이 프로젝트는 직무적성검사 시험을 준비하면서, 
보다 편리한 문제 풀이를 위해 제작하였습니다.

기본적으로 단일 웹페이지 기반으로 동작하는 
**학습/시험 보조용 인터랙티브 웹앱**이며,
다음 요소를 함께 담고 있습니다.

- 시험 시간 카운트다운·스톱워치 보기
- OMR 모양의 답안지에 1~5번 선택지 마킹
- 정답을 붙여넣고 자동 채점
- 정답/오답/미답 개수와 환산 점수 확인
- 어떤 문항에서 시간을 많이 썼는지 분석

온라인 주소:  
**https://5sangmin.github.io/OMR-pages/**

***

## 주요 기능

### 시험 시간 관리

- **실전 모드**: 30분, 15분, 10분, 5분 등 카운트다운 타이머
- **연습 모드**: 제한 없이 올라가는 스톱워치
- 시작, 중지, 초기화 버튼 제공
- 분석 버튼으로 시간 분석 패널 열기

### OMR 답안지

- 기본 30문항, 5지선다 OMR 형태
- 문항 수를 자유롭게 조정 가능
- 각 문항의 1~5번 버튼 클릭으로 답안 마킹
- 현재 선택한 문항 수 표시

### 정답 입력 및 채점

- 정답을 숫자로 입력하고 공백 또는 쉼표로 구분 가능
- 채점 버튼 한 번으로 결과 계산
- 정답 수, 오답 수, 미표기 수, 채점 대상 문항 수, 환산 점수 제공
- 정답과 사용자가 고른 답을 시각적으로 구분

### 오답·미답 분석

- 오답 문항과 미답 문항을 별도 목록으로 제공
- 어떤 문항을 틀렸는지 빠르게 확인 가능

### 문항별 시간 분석

- 분석 패널에서 문항당 평균 소요 시간 확인
- 총 경과 시간 확인
- 문항별 소요 시간을 많이 걸린 순서대로 정렬
- 요약 영역은 고정, 리스트 영역만 스크롤되도록 구성
- 기록이 없는 초기 상태에도 분석 영역 높이를 일정하게 유지

## 화면 구성

### 1. 시험 시간 카드

상단 영역에서 **실전 / 연습 모드**를 전환하고 타이머를 제어할 수 있습니다.

- 실전 모드: 카운트다운 방식
- 연습 모드: 스톱워치 방식
- `분석` 버튼: 응시 시간 분석 패널 열기

### 2. OMR 답안지 카드

가운데 영역에서 실제 OMR처럼 각 문항의 보기를 선택할 수 있습니다.

- 문항 수 조정 가능
- 답안 초기화 가능
- 정답 입력 영역으로 이동 버튼 제공

### 3. 정답 입력 및 채점 카드

정답을 입력한 뒤 채점할 수 있습니다.

- 공백/쉼표로 정답 입력
- 채점 및 입력 초기화 가능
- 배점 패널을 열어 정답/오답 계수 조정 가능

### 4. 채점 결과 카드

채점 후 결과 카드가 나타납니다.

- 정답, 오답, 미표기 개수 표시
- 환산 점수 표시
- 오답/미답 문항 상세 목록 제공

## 서비스 구성

전체 시스템 구조는 아래와 같은 구조로 이루어져 있습니다.

```text
.
├── index.html
├── manifest.webmanifest
├── sw.js
└── icons/
    ├── icon-192.png
    └── icon-512.png
```

- `index.html`: 메인 앱 화면과 기능이 들어 있는 파일
- `manifest.webmanifest`: 앱 설치(PWA)를 위한 설정 파일
- `sw.js`: 캐시 및 PWA 동작을 위한 서비스 워커 파일
- `icons/`: 앱 설치 시 사용되는 아이콘 파일

***

## 로컬에서 실행하는 방법

가장 간단하게는 **`index.html` 파일만 브라우저에서 열어도 기본 기능을 사용할 수 있습니다.**

### 가장 간단한 실행 방법

1. `index.html` 파일을 준비합니다.
2. 파일을 더블 클릭하거나 브라우저에서 엽니다.
3. 바로 OMR 답안지, 타이머, 채점 기능을 사용할 수 있습니다.

이 경우 일반 웹페이지처럼 동작하며, 핵심 기능은 사용할 수 있습니다.

### 참고

- `manifest.webmanifest`, `sw.js`, 아이콘 파일은 **PWA(앱 설치)** 기능을 위한 구성입니다.
- 따라서 **기본 기능만 사용하려면 `index.html`만 있어도 됩니다.**
- 다만 앱 설치, 홈 화면 추가, 캐시 기반 동작 같은 기능은 PWA 구성이 함께 있어야 제대로 사용할 수 있습니다.

## PWA 지원 내용

이 프로젝트는 웹앱을 스마트폰이나 PC에 **앱처럼 설치해서 사용할 수 있는 PWA 방식**도 지원합니다.

포함된 구성은 다음과 같습니다.

- `manifest.webmanifest`
- `icon-192.png`, `icon-512.png`
- `theme-color` 메타 태그
- `service worker` 등록 코드

즉, 일반 웹페이지로도 사용할 수 있고, 환경에 따라 앱처럼 설치해서 사용할 수도 있습니다.

### 앱 설치 방법

#### Desktop (Chrome / Edge)

1. 아래 주소를 엽니다.  
   **https://5sangmin.github.io/OMR-pages/**
2. 주소창 오른쪽의 설치 아이콘을 누르거나 브라우저 메뉴의 설치 항목을 선택합니다.
3. 설치 후 일반 앱처럼 독립 창으로 실행할 수 있습니다.


#### Android (Chrome)

1. Chrome에서 아래 주소로 접속합니다.  
   **https://5sangmin.github.io/OMR-pages/**
2. 브라우저 메뉴를 엽니다.
3. `홈 화면에 추가` 또는 `앱 설치`를 선택합니다.
4. 설치 후 홈 화면 아이콘으로 실행할 수 있습니다.


#### iPhone / iPad (Safari)

1. Safari에서 아래 주소를 엽니다.  
   **https://5sangmin.github.io/OMR-pages/**
2. 공유 버튼을 누릅니다.
3. `홈 화면에 추가`를 선택합니다.
4. 홈 화면에 생성된 아이콘으로 실행합니다.

***

## 사용 흐름 예시

1. 페이지에 접속합니다.
2. 시험 시간을 선택하고 `시작` 버튼을 누릅니다.
3. OMR 답안지에서 각 문항의 답을 선택합니다.
4. 필요하면 `분석` 버튼을 눌러 문항별 시간 분석을 확인합니다.
5. 정답 입력 영역에 정답을 붙여넣고 `채점` 버튼을 누릅니다.
6. 채점 결과와 오답/미답 문항을 확인합니다.
