# OMR 답안지 웹앱

브라우저에서 바로 실행할 수 있는 정적 HTML 기반 OMR 답안지 웹앱입니다. 시험 시간 관리, OMR 표기, 정답 입력, 채점, 오답/미답 확인, 간단한 메모/보조 도구 UI를 한 화면에서 사용할 수 있습니다.

## 주요 기능

- 시험 시간 설정 및 타이머 시작/초기화
- 실전 모드 / 연습 모드 전환
- 문항 수 설정 및 OMR 답안지 자동 생성
- 5지선다 OMR 답안 표기
- 정답 입력 후 즉시 채점
- 채점식 표시: `( 정답 × 1 ) - ( 오답 × 0.25 )`
- 오답 / 미표기 문항 구분 표시
- 채점 결과 요약(정답, 오답, 미표기, 환산 점수)
- 메모장 / 그림판 / 계산기 보조 패널 UI

## 파일 구조

```text
.
├─ index.html
└─ README.md
```

추후 PWA 또는 GitHub Pages 배포용으로 확장하려면 아래 구조를 사용할 수 있습니다.

```text
.
├─ index.html
├─ README.md
├─ manifest.webmanifest
├─ sw.js
└─ icons/
   ├─ icon-192.png
   └─ icon-512.png
```

## 실행 방법

### 로컬에서 바로 실행

1. `index.html` 파일을 저장합니다.
2. Chrome 브라우저에서 파일을 엽니다.
   - `Ctrl + O` 또는 `Cmd + O`
   - 또는 파일을 Chrome으로 드래그
3. 별도 설치 없이 바로 사용할 수 있습니다.

## GitHub Pages 배포 방법

GitHub Pages는 정적 사이트를 그대로 배포할 수 있으므로 `index.html`만 있어도 배포가 가능합니다.

1. GitHub에서 새 public repository 생성
2. `index.html`과 `README.md` 업로드
3. Repository의 **Settings → Pages** 이동
4. **Deploy from a branch** 선택
5. Branch를 `main`, Folder를 `/root`로 설정
6. 저장 후 몇 분 뒤 아래 주소로 접속

```text
https://사용자이름.github.io/저장소이름/
```

## 사용 방법

### 1. 시험 시간 설정

- 30분, 15분, 10분, 5분 중 선택하거나 직접 입력할 수 있습니다.
- `시작` 버튼으로 타이머를 시작합니다.
- `초기화` 버튼으로 시간을 리셋합니다.

### 2. 문항 수 설정

- 원하는 문항 수를 입력하고 `적용` 버튼을 누릅니다.
- 입력한 문항 수에 맞춰 OMR 답안지가 다시 생성됩니다.

### 3. 답안 표기

- 각 문항에서 1~5 중 하나를 선택합니다.
- 하단의 `선택 답안` 수치로 현재 표기 수를 확인할 수 있습니다.

### 4. 정답 입력 및 채점

- 정답 입력 영역에 정답을 숫자로 입력합니다.
- 입력 구분자는 쉼표 또는 공백을 사용합니다.
- 예: `1,2,3,4,5` 또는 `1 2 3 4 5`
- `채점` 버튼을 누르면 결과가 표시됩니다.

### 5. 채점 결과 확인

- 정답 문항 수
- 오답 문항 수
- 미표기 문항 수
- 채점 대상 문항 수
- 환산 점수
- 오답/미표기 상세 목록

## 채점 기준

현재 채점식은 아래와 같습니다.

```text
( 정답 × 1 ) - ( 오답 × 0.25 )
```

- 정답: 1점
- 오답: 0.25점 감점
- 미표기: 감점 없음

## 주의 사항

- 현재 버전은 정적 HTML 기반입니다.
- 로컬 파일(`file://`)로 열 경우 Chrome의 PWA 설치 버튼은 보이지 않을 수 있습니다.
- 앱처럼 설치하려면 HTTPS 환경(GitHub Pages, Netlify 등)에 배포하고 PWA 설정을 추가해야 합니다.
